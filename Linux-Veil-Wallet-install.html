How to download, install, and start the Veil QT Wallet and Snapshot using Linux

Requirements:

  1. A decent internet connection.
  2. At least 23.5GB (at the time of writing) of free storage to download the blockchain.

Download and starting the veil core wallet:

  1. Download the most up to date wallet from the Veil Github. Choose the appropriate version for your operating system (Windows, Mac, or Linux). I will be using Ubuntu 22.04 in this example.
  2. Download the wallet software on your computer. For this example I will choose x86_64-linux-binaries.zip.
  3. Once downloaded unzip the x86_64-linux-binaries and extract to an easy to find file location. I'm extracting to my desktop for simplicity.
  4. Navigate to the x86_64-linux-binaries. Right click on the veil-qt and select properties, permissions, and click Allow executing file as program. Then close window.
  5. Now right click the veil-qt icon and click run as program or use the terminal and start the wallet with ./veil-qt.

Wallet startup and Seed phrase:

  1. The wallet should open to the "Welcome to VEIL" screen.
  2. Use the default data directory. Click ok.
  3. Choose the language you wish. Click NEXT.
  4. Create wallet. Click NEXT.
  5. Get a pen paper or whatever and record and NEVER LOSE YOU SEED WORDS. Click Reveal seed phrase.
  6. Record the 24-word seed phrase. Only after you record your seed phrase. Press Next.
  7. Enter your 24-word seed phrase. Press Next.
  8. Let the wallet run for a few minutes. Then close the wallet.

Downloading and installing the snapshot:

  1. Go to Veil Tools and download the snapshot. Make sure you download the one closest to your region. Wait time will vary. Maybe go buy some Veil to stake while it's downloading.
  2. Extract the snapshot. I'm extracting to my desktop again.
  3. Open your file browser, go to home folder and click on the .veil folder. If .veil folder is not there, click the menu bar button (the one with 3 lines on the top nav bar. Near the minimize button at the top of the window) and check Show Hidden Files. Ctrl + h should work as well to show hidden files.
  4. Open the .veil folder. Inside delete ONLY the blocks, chainstate, indexes, and zerocoin folders.
  5. Copy and paste each folder one by one from the snapshot into the .veil folder. Blocks, zerocoin, indexes, and chainstate. Wait until they all appear in the folder. (You can click and drag them as well vs copy and paste.)
  6. Once it all transfers. Close the window.

Starting the wallet after snapshot:

  1. Right click on the veil-qt click run as a program or use the terminal and start the wallet with ./veil-qt.
  2. After starting, the wallet may not be fully synced to the current block, this may take additional time.

Get a basecoin address:

  1. If you get a pop-up screen stating your wallet is still synchronizing with the Veil network, click Hide. You should see a "Looks like you are new here" page. Lots of good information here about coin types, encryption, etcetera.
  2. To get a basecoin address you must click the gear icon on the right side tab. Click Advanced Options. Click the console tab. Then type getnewbasecoinaddress. Press enter. Then a bv address will appear. This is your new basecoin address.

Verify if synced:

  1. Check with the Veil Explorer to see when your wallet is fully synced with the network.

For more assistance visit the Veil discord or the Help Desk.
